# numberSystemConversion
Python program that converts different number systems such as: 

# DECIMAL CONVERSION
  Decimal - Hexadecimal
          - Binary
          - Octal
# HEXADECIMAL CONVERSION
  Hexadecimal - Decimal
              - Binary
              - Octal
# BINARY CONVERSION
  Binary - Decimal
         - Hexadecimal
         - Octal
# OCTAL CONVERSION
  Octal - Decimal
        - Hexadecimal
        - Octal
